ORM -> Prisma(Gerenciador de objetos relacionais).

Para implantar dados de csv no prisma, deve-se criar um arquivo de nome pde preferência.sql e aderir o seguinte comando:

	**LOAD DATA INFILE "C:/Users/Desenvolvimento.DESKTOP-CJ165MI/Desktop/SENAI/3DES/PROJETOS/AULA15(OSs)/docs/comentarios.csv"
	INTO TABLE comentario
	FIELDS TERMINATED BY ';'
	ENCLOSED BY '"'
	LINES TERMINATED BY '\r\n'
	IGNORE 1 ROWS;**

Após, abra o terminal do php admin, encontre o banco de dados que você está usando, abra o terminal sql e adicione os dados.
	Caso dê erro, provavelmente é erro de local.
