Como executar o programa: 
    * É necessário ter instalado o NodeJs, XAMPP, expo, nodemon.
    
    *A ordem de execução do programa é:
        FRONTEND:
            1.  Xampp (Start em MySQL)
            2.  cd back
            3.  npm i
            4.  nodemon
            5.  front index.html (Executar com live server)
        
        MOBILE: 
              1.  cd ..
              2.  cd app npm i
              3.  expo start --web
