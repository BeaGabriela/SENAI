On delete cascade -> Permite apagar alguma coisa assim que outra coisa com tal id seja apagado.
on uptade cascade -> Assim como o anterior, ele serve para alterar automaticamente assim que uma coisa com determinado id semelhante, receba alguma mudança.

Date_dub() -->  Serve para subtrair um intervalo de tempo ou data especificado para uma data especificada e, em seguida, retorna a data.

Sub_consultas -> Pode se consultar dados dentro de uma tabela, ao invés de digitar.

sql transacional --> Permitem a execução de uma sequência de operações como um bloco indivisível de forma a garantir a integridade dos dados em um ambiente com acesso concorrente.
Quando se mexe com mais de duas tabelas, é por que usa o mysql transação.
